# Rock, Paper, Scissors

## About:

**Hello and welcome!!**

This is my version of <q> Rock, Paper, Scissors </q>. For this project I employed the starter files available to me, I made changes and
styled to make it my own, creating a gradient for the background, changing the button, adding animations to set my project apart from others
and to help with UX and UI. I used arrow functions to familiarize myself more with them. In future project I want to do something similar with other
ways to code in order to expand my library of ways to solve problems.

### Technologies used:

- JavaScript.
- Bootstrap.
- HTML.
- CSS.

### How to use Front End:

- For the user to make a selection, all that needs to be done is hover over and click either: <q> rock </q>, <q> paper </q>, or <q> scissors </q>.
- Then the computer will make a selection.
- After that the selections will be compared, results will be tallied up and the scoreboard will be updated.
- To play again the user only has to click the <q> PLAY AGAIN </q> button and another round will begin.
- Player can restart a game as well by hitting the tab button until focus state is on <q> PLAY AGAIN </q> button;
  <q> custom styles have been set to know when its on focus state </q>. After hitting the return key another round will begin.
